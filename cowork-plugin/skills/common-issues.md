# Common Issue Templates

Pre-filled templates for the most frequently reported issues at Sweetgreen. Use these to create well-structured tickets quickly.

## WiFi / Network Issues

**Summary**: WiFi connectivity issue — [location]
**Category**: Network
**Subcategory**: Connectivity
**Description template**:
- Location: [store/office/floor]
- Issue: [unable to connect / intermittent drops / slow speeds]
- Affected devices: [laptop / phone / all devices]
- Started: [when did this begin]
- Number of people affected: [count]
- Troubleshooting attempted: [restarted device, forgot and rejoined network, etc.]

## Laptop Issues

**Summary**: Laptop [issue type] — [employee name or dept]
**Category**: Hardware
**Subcategory**: Computer
**Description template**:
- Device: [make/model if known]
- Issue: [won't turn on / blue screen / overheating / cracked screen / battery dead]
- How long: [when did it start]
- Is this blocking your work: [yes/no]
- Do you have a backup device: [yes/no]

## Password / Account Lock

**Summary**: Account locked — [system name]
**Category**: Access
**Subcategory**: Account
**Description template**:
- System affected: [Okta / email / specific application]
- Error message: [exact text if available]
- Last successful login: [approximate time]
- Have you tried self-service reset: [yes/no]

## Software Not Working

**Summary**: [App name] not responding — [brief description]
**Category**: Software
**Subcategory**: Application
**Description template**:
- Application: [name and version if known]
- Error message: [exact text or screenshot description]
- When it happens: [always / sometimes / after specific action]
- Impact: [can't do X, workaround is Y]
- Steps to reproduce: [1. Open app, 2. Click X, 3. Error appears]

## New Equipment Request

**Summary**: Equipment request — [item type] for [employee/dept]
**Category**: Hardware
**Subcategory**: Request
**Description template**:
- Item needed: [laptop / monitor / keyboard / mouse / headset]
- Reason: [new hire / replacement / upgrade]
- Needed by: [date]
- Employee: [name]
- Manager approval: [yes/pending]

## Printer Issues

**Summary**: Printer [issue] — [location]
**Category**: Hardware
**Subcategory**: Printer
**Description template**:
- Printer location: [floor/area]
- Printer name/model: [if visible on the device]
- Issue: [paper jam / not printing / offline / poor quality]
- Affects: [one person / entire floor]
